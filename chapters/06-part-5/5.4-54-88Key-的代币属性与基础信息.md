# 5.4 88Key 的代币属性与基础信息

88Key 被定义为 RWA 资产权益型代币，其核心功能是承接娱乐 RWA 资产的数字权益表达。 在88Key 的价值机制中，代币并不是脱离现实资产独立存在的流通符号，而是围绕 88 娱乐集团品牌资产、经营资产及真实现金流 所形成的权益载体。其价值基础来自 真实资产，运行基础来自真实经营，验证基础来自 真实现金流与持续信息披露机制。

从代币基础结构来看，88Key 的发行总量为 100,000,000 枚，发行网络为以太坊（Ethereum/ ETH）网络，发行机构为马来西亚·吉隆坡·88 娱乐集团，初始价格为 1 USD。 其底层资产包括 88 娱乐集团品牌资产、经营资产及真实现金流，价值锚定于 真实品牌资产与真实经营现金流。这一结构决定了 88Key 的代币属性并不只是链上流通工具，而是娱乐 RWA 资产权益进入数字资本体系后的基础承接单元。

在链上披露层面，88Key 已基于以太坊网络完成代币合约部署，并将合约地址纳入项目透明披露体系。88Key 合约地址为：0x9DCF492f3691d757F101541528802cE4E68bF066。 用户可通过 Etherscan 区块链浏览器 查看 88Key 的代币基础信息、合约状态及相关链上记录。Etherscan 当前页面显示该 Token 为 88key (88key)，最大总供应量为 100,000,000 88key，Token Contract 为该合约地址，并显示 Source Code Verified / Exact Match；同时页面显示No Contract Security Audit Submitted。因此，白皮书中应将其表述为 合约源码可公开查看与链上信息可查询，不应表述为已完成合约安全审计。


合约地址公开，是 88Key 构建链上透明机制的重要组成部分。 通过将合约地址、链上查看入口与源码状态纳入信息披露体系，88Key 进一步强化了代币基础信息的公开性、链上记录的可查询性以及数字权益表达的可追踪性，使用户能够更加直观地理解 88Key 的链上资产结构与权益承接基础。

需要明确的是，合约公开查询、源码可查看与链上信息可验证，并不构成收益承诺、价格承诺或安全审计结论。 88Key 的长期价值，仍将建立在 真实娱乐资产、持续经营能力、真实现金流验证、信息披露机制、治理规则执行与市场参与结构 之上。
